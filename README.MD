# ComfyUI Image Pin

### 简介

ComfyUI Image Pin 是一个 ComfyUI 的自定义节点，允许您在 ComfyUI 工作流中“钉住”图像。这对于在处理图表的其他部分时保持参考图像可见非常有用。

### 安装

1. 导航到您的 ComfyUI 安装目录下的 `custom_nodes` 文件夹。
2. 克隆此仓库：
   ```bash
   git clone https://github.com/CheNing233/ComfyUI_Image_Pin.git
   ```
3. 重启 ComfyUI。

### 使用方法

搜索 `ImagePin` 节点
